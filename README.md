# AgriData-Explorer-Understanding-Indian-agriculture-with-EDA
# Objective:
Build a unified data analysis and visualization platform for Indian agriculture by simplifying fragmented datasets and bringing them into one structured system. The platform reveals insights on crop production, yield trends, and regional variations to support informed decisions for farmers, policymakers, and researchers.
# Technologies:
Python | SQL | Power BI | Data Cleaning | Data Analysis | Data Visualizations.
